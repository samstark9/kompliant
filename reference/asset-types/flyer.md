# Asset type: Flyer

Printed or digital one-page recruiting piece. The income table lives here more than anywhere else, which makes the flyer the highest-stakes asset for the False Claims and Empty Perceptional Promises families.

## Where flyers fail first

1. The income table: bare figures, outlier-as-typical, compressed timelines, missing small type (A5, A6, A7). The canonical correction is example 1 in `examples.md`.
2. "Provide" language in the training and leads bullets (A1).
3. "Free" on the lead-magnet or training line (A10).
4. Guarantee-adjacent closers: forever language, unhedged effort language (A9).

## Review emphasis, in order

1. Income table against the profile table: figures, timelines, hedging, asterisks.
2. Small type: present, same page, complete year, named source, "not guaranteed." Flag type size and placement for layout check.
3. Term sweep per `reference/term-banks.md`.
4. Titles against the profile ladder.
5. Stability claims: company stability only, sourced (B4, A6).
6. Imagery and branding (A13, A14).

## Asset notes

- A flyer is reviewed as a page: claims and their small type must land on the same page by definition. If the draft is multi-page, each page carries its own proof trail.
- Common structure runs hook, opportunity, income table, stability proof, incentives, call to action. Structure is craft (Class C); compliance applies regardless of order.
